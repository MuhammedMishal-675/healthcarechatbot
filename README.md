Healthcare AI Chatbot

Project Overview

This project is a Healthcare AI Chatbot built using Python, Flask, and NLP techniques.
It provides basic health-related guidance based on user queries such as symptoms, medicines, diet, lifestyle, and mental health.

---

Features

- Symptom-based guidance
- Medicine information
- Diet suggestions
- Lifestyle tips
- Mental health support
- First aid guidance
- Chat logging

---

Technology Used

- Python
- Flask
- HTML/CSS
- Sentence Transformers

---

 How It Works

User → Flask Backend → AI Engine → Response → UI

---

 Project Structure

- app.py → Backend
- chatbot/engine.py → AI logic
- templates/index.html → UI
- logs/ → Chat history

---

How to Run

1. python -m venv venv
2. venv\Scripts\activate
3. pip install -r requirements.txt
4. python app.py

Open: http://127.0.0.1:5000/
